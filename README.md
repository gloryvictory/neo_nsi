# НСИ Viewer — Просмотр справочников PostgreSQL

React-приложение для просмотра и экспорта таблиц-справочников из PostgreSQL.

## Установка и запуск

### 1. Настройка подключения к БД

Скопируйте файл конфигурации и заполните данные:

```bash
cp .env.example .env
```

Отредактируйте `.env`:

```env
DB_HOST=192.168.1.100
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=your_password
DB_NAME=neo_nsi
DB_SCHEMA=public
TABLE_PREFIX=Справочник
SERVER_PORT=3001
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Запуск в режиме разработки

```bash
npm run dev
```

- **Frontend**: http://localhost:5173
- **API**: http://localhost:3001/api

### 4. Production-сборка

```bash
npm run build
npm start
```

Приложение будет доступно на http://localhost:3001

## Функциональность

- **Список справочников** — отображение таблиц, начинающихся с префикса "Справочник"
- **Поиск** — мгновенный фильтр по списку таблиц и глобальный поиск по данным
- **Фильтры по колонкам** — индивидуальные фильтры для каждой колонки таблицы
- **Сортировка** — клик по заголовку колонки для сортировки (ASC/DESC)
- **Пейджинг** — выбор количества строк на странице (10/25/50/100)
- **Экспорт в Excel** — выгрузка текущих данных (с учётом фильтров) в .xlsx
- **Сворачиваемый сайдбар** — компактный режим для больших мониторов

## Стек технологий

- **Frontend**: React 18 + Vite
- **Backend**: Express.js
- **БД**: PostgreSQL (через pg)
- **Excel**: ExcelJS
